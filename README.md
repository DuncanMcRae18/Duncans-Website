# Duncan McRae - Personal Portfolio Website

A clean, modern, JS-oriented personal portfolio website built for learning JavaScript and web development.

## 📁 Project Structure

```
PortfolioWebsite/
├── src/                          # Source code directory
│   ├── index.html                # Main HTML file (edit your content here)
│   ├── components/               # Reusable HTML components
│   ├── scripts/
│   │   ├── index.js              # Main JavaScript file
│   │   ├── nav.js                # Navigation logic
│   │   ├── scroll.js             # Smooth scrolling
│   │   └── interactions.js       # User interactions
│   └── styles/
│       ├── style.css             # Main stylesheet
│       ├── components.css        # Component-specific styles
│       └── responsive.css        # Mobile/responsive styles
│
├── public/                       # Static assets
│   ├── images/                   # Project images and photos
│   └── documents/                # PDFs, resumes, proposals
│
├── CNAME                         # Custom domain configuration
├── sitemap.xml                   # SEO sitemap
└── README.md                     # This file
```

## 🚀 Getting Started

### 1. Edit Your Content
- **HTML**: Update `src/index.html` with your name, bio, projects, experience
- **Styles**: Customize in `src/styles/`
- **JavaScript**: Add interactivity in `src/scripts/`

### 2. Add Media
- Images: `public/images/`
- Documents: `public/documents/`

### 3. Local Development
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server .
```
Visit `http://localhost:8000/src/`
